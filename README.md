# Calculadora-Macros

Este proyecto tiene como Objetivo calcular la ingesta total de un Usuario segun
su Objetivo Fitness y Frecuencia de Entrenamiento, entregando los siguientes
valores:

- Proteinas (gr)
- Carbohidratos (gr)
- Grasas (gr)
